# Plan — ALKU II horizontal credit scroller (`alku2`)

Branch `build/20-alku2` from HEAD. Mirror the `alku1`/`forest` part split: pure unit-tested logic + GPU
nodes + an `Effect` on a fixed-timestep accumulator with `setMode`, rendering into the supplied target.

## Steps

1. **Design + plan docs** (this + the design doc). ✔
2. **Assets** — copy `FONA.UH`, `HOI.U` into `apps/lab/public/pics/` and `src/alku2/__fixtures__/`. ✔
3. **`text-buffer.ts` + test** — `addText` ports `MAIN.C addtext`: centred chunky glyph stamp into the
   352×186 `tbuf`, ink level → `level*0x40`. Test: a known string lands centred, ink bytes correct, widths
   match `font.measure`.
4. **`scroll.ts` + test** — credit cards (verbatim `MAIN.C:103-128`), `SCRLF=9` cadence, `scrollAt(frame)`
   → `{ scroll, card }` deterministic + looping. Test: cadence (one step per 9 frames), card schedule, loop.
5. **`copper.ts` + test** — `backdropOffset(frame)` (HOI horizontal pan, wraps over the 640 source width) and
   `sampleBackdropRow`. Test: advance + wrap.
6. **`compose.ts` + test** — `composeFrame(dst, hoi, tbuf, scroll, off)`: HOI window + scrolled text band.
   Test: backdrop fills the field; text translates with the scroll offset; OR-merge over backdrop.
7. **`nodes.ts`** — `RasterSurface` (index→6-bit LUT→sRGB DataTexture→Blit), Nearest/Linear toggle. Same as
   alku1.
8. **`alku2.ts`** — the `Effect`: load FONA+HOI via `decodeU`/`loadFona`, build `tbuf`s, 70 Hz accumulator,
   `setMode`, render into supplied target, dispose teardown.
9. **`index.ts`** — re-export `Alku2`, `LookMode`, and the pure helpers (so tests import directly; the part
   stays UN-WIRED — `packages/parts/src/index.ts` is NOT touched).
10. **Verify** — `pnpm lint`, `pnpm typecheck`, `pnpm test`, `pnpm build` all green. STATUS doc. Commit.

## Constraints

- TS strict + noUncheckedIndexedAccess + exactOptionalPropertyTypes + verbatimModuleSyntax; no `!`; explicit
  `.js` imports; `import type` for types. Biome 2-space/single-quote/width-100.
- C int division truncates toward zero — use `Math.trunc`/`>>`/`| 0` to match.
- No engine edits, no `packages/parts/src/index.ts` edit, no lab edits. Engine APIs used read-only:
  `decodeU`, `loadFona`, `BitmapFont`, `Blit`, the `Effect`/`RenderTarget`/`FrameContext` types.
